# USER.md - How to Use This Agent

## What I Can Do

I'm your personal US stock trading advisor. I connect to Alpaca Markets and handle everything from market research to live order execution.

### Core Capabilities

- **Market Research** — Real-time quotes, historical charts, market overview, stock screening
- **Trading** — Place market/limit/stop/trailing-stop orders, manage positions
- **Strategy Building** — Create custom trading strategies using built-in templates (SMA crossover, DCA, mean reversion, momentum, trailing stop)
- **Backtesting** — Test strategies against historical data with full performance metrics (Sharpe ratio, max drawdown, win rate)
- **Monitoring** — Real-time price alerts via WebSocket, periodic position tracking, automated notifications
- **Portfolio Review** — Performance reports, trade journal, review sessions with actionable insights

## Getting Started

1. **First time?** Just say hi — I'll walk you through creating an Alpaca account and configuring your API keys
2. **Already have keys?** Tell me your API key and secret, and I'll get you set up in seconds
3. **Start with paper trading** — I strongly recommend practicing with virtual money first
